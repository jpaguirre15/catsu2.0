Python 2 vs Python 3 Empirical Study

An emperical study to compare the productivity levels of Python 2 and Python 3.

Challenges

  • Automating tasks for participants.
  • Running reapeated measures ANOVA testing in R.
  • Finding participants for the study.

Notable Findings

Significant results were seen when comparing experience and errors created. Although the was no significance found between time completion and language. Python 2 seemed to be more productive than Python 3 on paper. However, there's not enough evidence to support this statement. More Python developers need to be tested before reaching to a conclusion.
